Conversation and Memory Management
==================================

This module defines data models for conversation telemetry and both synchronous and asynchronous memory managers
backed by :postgresql:`PostgreSQL <about>`. It allows persisting conversation turns, retrievals, LLM telemetry, and user feedback.
